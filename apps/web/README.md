# apps/web

Minimal Next.js App Router frontend scaffold for BiasharaMind.

## Included

- App Router structure with route groups for marketing, auth, and dashboard flows
- Placeholder pages for the V1 product journey
- Shared layouts and lightweight UI shells for future feature work
- Minimal Next.js, TypeScript, and CSS configuration

## Scripts

- `npm run dev`: start the development server
- `npm run build`: create a production build
- `npm run start`: start the production server
- `npm run lint`: run Next.js linting

## Notes

- No authentication, API integration, or business logic is implemented yet.
- Pages are server components by default.
- Styling is intentionally minimal and production-leaning.
